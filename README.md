# remote

The remote code

## Setup

Things that need to be installed:

- pigpio
- RPi.GPIO: ` sudo apt-get install python-rpi.gpio python3-rpi.gpio`

## Pin Numberings

| Pin | Meaning                |
|-----|------------------------|
| 8   | Power button           |
| 10  | Brightness up button   |
| 12  | Brightness down button |
| 11  | Preset 1               |
| 13  | Preset 2               |
| 15  | Preset 3               |
| 29  | RGB (R)                |
| 31  | RGB (G)                |
| 33  | RGB (B)                |
